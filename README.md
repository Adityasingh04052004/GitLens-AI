# GitLens AI – AI-Powered GitHub Codebase Assistant

GitLens AI is a production-quality full-stack AI project that clones public GitHub repositories, analyzes their structure, automatically detects the technology stack, and provides AI-generated code summaries and interactive codebase chat.

## Project Structure
- `client/`: React + Vite frontend application.
- `server/`: Node.js + Express backend application.
- `README.md`: Root documentation.
